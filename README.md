# Manage Landing Page

A modern, responsive landing page built using **HTML**, **CSS**, **JavaScript**, and **Tailwind CSS**. This project showcases a clean business landing page design featuring a responsive navigation menu, hero section, features section, testimonials, call-to-action area, and footer.

## ⚙️ Mobile Menu Functionality

The mobile navigation menu is implemented using JavaScript.

```javascript
const btn = document.getElementById('menubtn');
const nav = document.getElementById('menu');

btn.addEventListener('click', () => {
    nav.classList.toggle('flex');
    nav.classList.toggle('hidden');
});
```
